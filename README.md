# h2touch Project Management Dashboard

Веб-приложение для управления проектами и дорожной картой развития экосистемы h2touch на базе системы "Здоровье 2.0" Артёма Хачатряна.

## 🚀 Функциональность

- **📊 Dashboard** - обзор всех проектов и их статуса
- **📅 Calendar** - календарь мероприятий и дедлайнов  
- **🎯 Goals** - система целей и задач
- **📈 Progress** - отслеживание прогресса проектов
- **📱 Responsive** - адаптивный дизайн для всех устройств

## 🛠 Технологии

- Pure HTML/CSS/JavaScript (без фреймворков)
- Chart.js для графиков и диаграмм
- Responsive design
- Local storage для сохранения данных

## 📖 О системе "Здоровье 2.0"

Приложение построено вокруг революционной системы здоровья из 5 взаимосвязанных сфер:

1. **Тело** - физическое состояние
2. **Образ жизни** - ежедневные привычки
3. **Состояние** - эмоции и энергия
4. **Окружение** - социальная среда
5. **Аутентичность** - соответствие внутренних ценностей внешнему проявлению
